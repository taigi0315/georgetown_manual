# TASK-006: 카드 이미지 크기 확대 및 인코딩 깨짐 해결

## 1. 개요
- 일부 카드 이미지 URL 내의 `&` 문자가 HTML 엔티티인 `&amp;`로 치환되어 이미지가 제대로 보이지 않고 깨지던 문제를 해결합니다.
- 카드 요약(summary) 영역의 썸네일 이미지가 너무 작아 시각적 인지가 어렵다는 피드백에 따라, 썸네일 크기를 확대하여 가독성을 높입니다.
- 카드를 열었을 때 상단에 표시되는 본문 히어로 이미지의 세로 비율도 조율하여 이미지를 더 선명하고 시원시원하게 보이도록 개선합니다.

## 2. 세부 작업 항목
- **이미지 URL 엔티티 깨짐 일괄 복구**:
  - `index.html` 및 `agent.html` 소스 파일 내 `src` 속성에 있는 `&amp;`를 순수 `&`로 일괄 치환하여 이미지 쿼리 깨짐 방지.
- **썸네일 및 본문 이미지 크기 확대 (CSS 조율)**:
  - 썸네일 이미지(`.card-thumbnail-img`) 크기 상향: `48px * 48px` ➡️ **`72px * 72px`** (또는 **`80px * 80px`**).
  - 모서리 둥글기(`border-radius`)를 `8px`로 부드럽게 조정.
  - 본문 상세 이미지(`.card-hero-img`) 비율 조정: `aspect-ratio: 16 / 7` ➡️ **`16 / 9`** (또는 `height: 220px`)로 세로 폭을 확장하여 이미지가 시원하게 보이도록 개선.
- **AI Agent 안내 문서(agent.html) 갱신**:
  - `agent.html` 내의 CSS 스타일 룰 규격 및 버전을 **v2.6**으로 동기화 업데이트.
